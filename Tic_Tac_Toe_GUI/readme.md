# 🎮 Tic-Tac-Toe GUI (Python/Tkinter)

A fully interactive, graphical Tic-Tac-Toe game built from scratch using Python and the Tkinter library. This project features a complete user flow, from player registration to customizable game symbols and automatic win/tie detection.

## ✨ Features

* **Multi-Screen Interface:** Navigates smoothly through a sequence of frames (Login -> Confirmation -> Symbol Selection -> Gameplay).
* **Player Registration:** Allows two players to input their names for a personalized gaming experience.
* **Customizable Symbols:** Players can choose their preferred game markers (e.g., `X/O`, `@/#`, `$/%`, `*/&`).
* **Smart Game Logic:** 
  * Alternating turn management.
  * Automatic win detection (rows, columns, and diagonals) with visual highlighting.
  * Tie detection when the board is full.
* **Session Management:** Includes "Reset" (clear the board) and "Restart" (enter new names) functionality.
* **Embedded Assets:** The application icon is encoded directly in Base64 within the script, eliminating the need for external image files.

## 🛠️ Prerequisites

To run this project, you will need Python 3 installed on your system along with the `Pillow` library for image processing.

```bash
pip install Pillow
```
