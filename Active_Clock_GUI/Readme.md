# 🕒 Python Digital Clock

## 📌 Project Overview
A simple, lightweight, and real-time digital clock built using Python's standard GUI library, `tkinter`. This desktop application dynamically displays the current time (in a 12-hour format with AM/PM), the day of the week, and the local date, updating automatically without blocking the main event loop.

## ✨ Features
* **Real-Time Timekeeping:** Continuously updates the time using the `.after()` method for non-blocking UI refreshes.
* **Comprehensive Display:** Shows the exact time, day, and date in a clear, high-contrast visual format.
* **Zero External Dependencies:** Built entirely with Python's standard built-in libraries (`tkinter` and `time`), meaning no `pip install` is required.

## 🛠️ Prerequisites
* **Python 3.x** installed on your system.
* *Note: `tkinter` comes pre-installed with standard Python distributions on Windows and macOS. Linux users may need to install it via their package manager (e.g., `sudo apt-get install python3-tk`).*

## 🚀 How to Run
1. Save the code to a file named `clock.py`.
2. Open your terminal or command prompt.
3. Navigate to the directory where you saved the file.
4. Run the script using the following command:
   ```bash
   python clock.py
   ```
